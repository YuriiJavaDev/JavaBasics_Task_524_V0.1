## Create a JFrame window with:

**🔷 The overall window structure should use `BorderLayout`**

Place:

---

**🟦 NORTH**

Application Title:

- `JLabel` with the text: **"Login System"**
- the text should be centered
- the font can be bold (optional)

---

**🟨 CENTER**

The center section should contain a login form, styled using `FlowLayout`:

- `JLabel("Username")`
- `JTextField(10)`
- `JLabel("Password")`
- `JPasswordField(10)`
- `JButton("Login")`

👉 All elements should be inline and wrap automatically (FlowLayout)**

---

**🟩 SOUTH**

- `JLabel` with the text: "Status: Not logged in"
